Sistema de Fatura

Desenvolvimento de um sistema de faturação com acoplamento fraco, projetado para criar e gerenciar faturas de forma eficiente por meio de uma API REST.

  • Mensageria com RabbitMQ: Utilização do RabbitMQ para comunicação assíncrona e confiável entre os serviços, promovendo maior eficiência e resiliência no processamento de tarefas.
  • Containerização com Docker: Toda a aplicação foi containerizada, garantindo portabilidade e facilidade de implantação em diferentes ambientes.
  • Desenvolvimento com TypeScript
  • Backend em Node.js com Express.js: Fornecendo uma API REST robusta e performática para o gerenciamento de faturas.
